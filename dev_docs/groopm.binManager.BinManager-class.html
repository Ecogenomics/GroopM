shouldMerge(self, bin1, bin2, ignoreCov=False, ignoreMer=False, merTol=0, confidence=0.95, verbose=False)

source code 

Determine whether its wise to merge two bins

Perfoms a one-way anova to determine if the larger bin would be significantly changed if it consumed the smaller

OR does a tolerance test on kmerNormPC1. We assume that bin1 is larger than bin2
